VS Code :

Visual Studio Code (VS Code) is a free, cross-platform code editor developed by Microsoft

It is lightweight yet powerful, offering features tailored for developers to write, debug, and optimize code efficiently.

Vs Code Features:

Code in any Language: write code your favorite language like JavaScript, Python, Java, c++ and more.

Version Control:Version control Built-in support for git and many other source control providers.

Debugging:Debug your code without leaving your editor.

Integrated Terminal: Allows running command-line tasks without leaving the editor.

Extensions: A vast marketplace for plugins to extend functionality, including linters, debuggers, and theming.

Customization: Highly customizable with themes, keybindings, and workspace settings.

Emmet Shortcuts

Emmet Symbol Description
! Expands to the html5 template.
TagName Expand the complete the tag
> It is used to create child elements within a parent element
+ It is used to create sibling elemetns.
^ Moves up one level in the hierarchy, allowing you to add sibling elements to parent
* It is used to create multiple elements.
{} This Symbol is used to write content between the tags
[] This is used to write attributes in the elements.

Keyboard Shortcuts

Short Cut Description
Ctrl + X Cut the selected text or content
Ctrl + C Copy the selected text or content
Ctrl + V paste the selected text.
Ctrl + S Save the current file.
Ctrl + A select all content in the current file.
Ctrl + O Opens the selected file.
Ctrl + Z Undo, we can instantly reverse your last action.
Ctrl + Y The Redo function restores any action that was previously undone with Undo
Ctrl + Shift + P Open Command Palette
Ctrl + Shift + N Its open New window
Ctrl + Shift + W Close window
Ctrl + Shift + Alt + Down Add cursors to multiple lines for simultaneous editing
Alt + Down Arrow: Move the selected line or block of code one line down
Alt + Up Arrow: Move the selected line or block of code one line up.
